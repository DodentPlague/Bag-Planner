# Bag Planner
A proof of concept budgeting/e-transfer app

<img width="320" height="660" alt="Screenshot 2026-02-23 011559" src="https://github.com/user-attachments/assets/5aae02ed-644d-48b9-8cb9-46f3b91cb614" />

# Features
- Budget management
- Connections for transferring money to friends/roommates/family
- Mobile and web app via React Native / Expo

Currently the app is not actually attached to any real banking system, so all money is fake.

# Technologies
- Frontend in JavaScript (React Native / Expo)
- Backend with Java (Netty) and SQL database (SQLite)
